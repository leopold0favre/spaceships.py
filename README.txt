
A lancer avec "python3 spaceships.py"

Mon tout premier projet dans le domaine de la programmation.

Il s'agit d'un jeu développé en python en utilisant uniquement la bibliothèque pygame.

Le code est illisible, il y a beaucoup de code redondant, et celui-ci n'a pas été mis à jour depuis 2017.
Le but n'est pas ici de montrer une quelconque compétence, mais de garder en mémoire la motivation du
développeur très débutant que j'étais en classe de terminale !

Le projet sera laissé en l'état. J'envisage de peut-etre le recréer de A à Z avec mes nouvelles compétences un jour.

Un bug notable qui n'était pas présent à l'époque est le freeze régulier de l'affichage, et j'ai beau essayer je n'
arrive pas à le corriger à cause de l'illisibilité du code du moi du passé. Une belle leçon de programmation.
Peut-être est-ce du au fait que le programme a été testé sous python 2.7 à l'époque.
